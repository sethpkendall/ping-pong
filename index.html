Ping-Pong Counter

Welcome to the Ping-Pong Counter! When given a number this program will count up to the inputted number with some special insertions. These special insertions are:

  • "Ping" for numbers divisible by 3.
  • "Pong" for numbers divisible by 5.
  • "Ping-Pong" for numbers divisible by 15.

Please input a positive integer below.
